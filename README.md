# 百度输入法超级皮肤编辑器
百度输入法提供了一个超级皮肤功能，其功能不只是改变外观，而是可以做到修改键位，自定义键盘布局，可以轻易实现其他输入法难以实现的功能。
但是，官方并未提供新版编辑器，目前网络上能够找到的编辑器版本非常久远（大约2015年），而且并不支持现在的多数配置，新版配置文件中新增一些配置项，旧编辑器无法解析，导致无法使用。
本项目旨在解决这一痛点，提供高度自定义的修改功能，同时尽可能降低使用门槛，减少学习时间。

### 项目说明
- 由于能力和python gui库等原因，本项目前端使用由Next.js构建的WebUI，详见[百度输入法超级皮肤编辑器（前端依赖）](https://github.com/gozhuimeng/KeyEditor-client)
- 由于时间精力有限，该项目将逐步开发完善功能，并非一次提供完整可用版本
